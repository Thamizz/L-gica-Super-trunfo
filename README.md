# L-gica-Super-trunfo
**Nível Aventureiro**
O programa permite que o usuário cadastre duas cartas do jogo Super Trunfo.
Logo em seguida é possível escolher um atributo através de um menu para comparar entre as cartas (área, PIB, Densidade...).
Por fim o programa mostra o resultado da comparação e as informações relevantes.

Caso o usuário escolha uma opção inexistente do menu, o programa exibirá a seguinte frase "Opção inválida!".
